Cómo crear un sensor

Para crear un sensor haz clic en el botón Nuevo sensor ubicado en la lista de sensores. El proceso básico requiere solo dos datos: el nombre del sensor y la encuesta que utilizará.

1

Hacer clic en Nuevo sensor

El botón se encuentra en la barra superior de la sección Sensores.

2

Completar el formulario básico

Ingresa el nombre del sensor y selecciona la encuesta a utilizar.

3

Guardar

El sensor quedará creado y en ejecución de inmediato.

Figura 2 — Formulario de creación básico.
Campo Obligatorio Descripción
Título Sí Nombre con el que se identificará el sensor en QServus.
Encuesta Sí Cuestionario de preguntas que utilizará el sensor.
💡
Si necesitas configurar opciones adicionales como fecha de cierre, diseño de banner, redes sociales o geolocalización, puedes hacerlo desde las Opciones avanzadas antes de guardar.
