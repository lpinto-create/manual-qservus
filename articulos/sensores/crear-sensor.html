Cómo crear un sensor

Haz clic en el botón Nuevo sensor ubicado en el listado de sensores.

Figura 4.6 — Botón nuevo sensor

Formulario básico

El formulario básico de creación requiere solo dos datos:

  • Nombre del sensor: identificador interno para reconocerlo en el listado.
  • Encuesta a utilizar: selecciona la encuesta publicada que quieres asociar al sensor.
Figura 4.7 — Formulario de creación básico
💡
Para configurar opciones adicionales como diseño visual, comportamiento de respuestas o datos solicitados, accede a las Opciones avanzadas antes de guardar.

Después de crear

Una vez guardado, el sensor queda en estado En ejecución y comienza a aceptar respuestas inmediatamente. Accede al detalle del sensor para obtener el código QR, URL y widget HTML.
