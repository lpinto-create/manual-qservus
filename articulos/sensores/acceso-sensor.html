Pestaña Acceso

La pestaña Acceso del detalle del sensor concentra todas las opciones para distribuir la encuesta: código QR, URL única, widget HTML y más.

Figura 1 — Pestaña Acceso con todas las opciones de distribución.
Opción Descripción
Pre-visualizar encuestaAbre la encuesta en una nueva pestaña. No guarda respuestas ni registra el acceso.
Ver código QRMuestra la imagen del código QR descargable para personalizar el diseño.
Descargar PDFGenera el código QR ya diseñado, listo para imprimir.
Descargar PDF personalizadoPermite definir opciones de diseño antes de generar el archivo.
Descargar PDF plegableGenera el QR en formato plegable, ideal para stands o mesas.
Compartir sensorEnvía por correo la URL del sensor.
Widget HTMLGenera el código HTML para insertar el sensor directamente en tu sitio web, con previsualización del diseño.
Enviar widget del sensorEnvía por correo el código HTML del widget.
Vanity URLDirección web personalizada y fácil de recordar para el sensor (ej. https://qserv.us/DvaCsz/). Para solicitarla escribe a comercial@redcalidad.com.
