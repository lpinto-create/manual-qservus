¿Qué es el reporte web?

El reporte web genera automáticamente un sitio web con los resultados de la encuesta seleccionada. Cada reporte tiene una URL única que puedes compartir con quienes necesiten revisar los resultados, sin necesidad de descargar ningún archivo.

Cómo generar el reporte

Desde las opciones de la encuesta, selecciona Reporte Web. El sitio se generará automáticamente y recibirás la URL para compartir.

💡
El reporte web es ideal para compartir resultados en tiempo real con equipos o clientes externos. Si necesitas un documento descargable, usa el Reporte PDF.
