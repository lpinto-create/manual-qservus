¿Qué es el reporte web?

Al seleccionar Reporte Web desde el menú de opciones de la encuesta, el sistema genera automáticamente un sitio web con el informe de resultados. Este sitio tiene una URL única que puedes compartir con cualquier persona sin que necesite acceso a QServus.

Figura 3.67 — Informe de respuestas web

¿Para qué usarlo?

  • Compartir resultados con stakeholders o clientes que no tienen cuenta en QServus
  • Presentar los datos de satisfacción en reuniones sin necesidad de exportar archivos
  • Mantener un enlace actualizado con los resultados acumulados de la encuesta
💡
El reporte web se actualiza automáticamente con los nuevos resultados. No es necesario regenerarlo cada vez que lleguen nuevas respuestas.
ℹ️
El reporte web consolida los resultados de todos los sensores y campañas que usan la encuesta. Para reportes individuales por canal, usa Reportes de campañas o Reportes de sensores.
