Texto simple

Permite que el cliente ingrese una respuesta corta y directa en un campo de texto de una sola línea. Ideal para nombre, correo, número de orden u otras respuestas breves y específicas.

Figura 3.22 — Pregunta tipo texto simple

Texto multilínea

Proporciona un área de texto amplia para respuestas largas. Ideal para capturar opiniones, comentarios, sugerencias o cualquier respuesta que requiera más extensión.

Figura 3.23 — Pregunta tipo texto multilínea

Fecha

Muestra un selector de fecha para recopilar información como fecha de nacimiento, fecha de compra, inicio o fin de una experiencia, etc.

Figura 3.24 — Pregunta tipo fecha

Foto

Permite que el cliente suba una imagen o tome una fotografía directamente desde su dispositivo como respuesta a la pregunta.

Figura 3.25 — Pregunta tipo foto

Archivo

Permite adjuntar archivos directamente a la respuesta: fotos, documentos, videos u otros. Útil cuando se necesita evidencia documental junto con la respuesta.

Figura 3.26 — Pregunta tipo archivo

HTML

Permite diseñar una pregunta personalizada usando código HTML. Ofrece total flexibilidad para crear elementos no disponibles en los tipos estándar: campos especiales, botones personalizados, listas desplegables, etc.

⚠️
El tipo HTML requiere conocimientos de desarrollo web. Su uso incorrecto puede afectar la visualización de la encuesta.
