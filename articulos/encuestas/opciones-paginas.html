Opciones disponibles

Las páginas se pueden reordenar, ocultar, editar y eliminar. Para mover las páginas se utilizan los íconos de flechas ubicados en los extremos.

Figura 3.17 — Páginas — opciones de configuración
Opción Descripción
Mover hacia la izquierda Desplaza la página actual hacia la izquierda. No se muestra si es la primera página.
Ocultar encabezado No muestra el título y descripción de la página al encuestado.
Editar Abre el formulario de edición de la página actual.
Eliminar Elimina la página actual y todas sus preguntas. Esta acción no se puede deshacer.
Mover hacia la derecha Desplaza la página actual hacia la derecha. No se muestra si es la última página.

Crear nueva página

Para crear una nueva página haz clic en el ícono de añadir ubicado al costado izquierdo del ícono de vista previa. Esto abrirá un modal con las opciones de configuración de la nueva página.

Figura 3.18 — Modal para agregar página
⚠️
Eliminar una página elimina también todas las preguntas que contiene. Esta acción no se puede deshacer.
