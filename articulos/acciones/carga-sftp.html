¿Cómo funciona?

Esta acción envía las respuestas de la encuesta de forma automática a un servidor SFTP configurado, cada vez que un encuestado completa la encuesta. Es ideal para integraciones con sistemas externos que procesan los datos de satisfacción.

Configuración

Crea una nueva acción de tipo SFTP y completa el formulario con los datos de conexión y credenciales del servidor destino.

Figura 1 — Formulario de configuración de carga de respuestas a SFTP.
ℹ️
Para configurar esta acción necesitas los datos de acceso al servidor SFTP: host, puerto, usuario y contraseña o llave SSH.
