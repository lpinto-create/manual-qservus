¿Cómo funciona?

Esta acción permite enviar automáticamente las respuestas de la encuesta a un servidor SFTP configurado por tu organización. Es útil para integraciones con sistemas internos que requieren recibir los datos de forma estructurada y automatizada.

Configurar la acción

Para crear una acción de carga a SFTP, selecciona el ícono de creación en la lista y completa el formulario con los datos del servidor SFTP de destino:

1

Nombre de la acción

Asigna un nombre identificador para esta configuración.

2

Datos del servidor SFTP

Ingresa el host, puerto, usuario y contraseña del servidor SFTP al que se enviarán las respuestas.

3

Ruta de destino

Define el directorio del servidor donde se depositarán los archivos de respuestas.
