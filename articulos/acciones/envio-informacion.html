¿Cómo funciona?

Esta acción permite definir condiciones para mostrar al respondente información personalizada al finalizar la encuesta. Dependiendo de las alternativas que haya seleccionado, se le puede enviar un mensaje, uno o varios enlaces, o archivos adjuntos relevantes.

Figura 8.12 — Listado de acciones de envío de información

Crear una acción de envío de información

En la lista de acciones de envío de información, selecciona el ícono para crear una nueva acción y completa el formulario de configuración.

Figura 8.13 — Formulario de configuración de envío de información

Agregar condiciones

Las condiciones definen cuándo se mostrará la información. Para cada condición debes especificar qué pregunta y qué alternativa debe haber seleccionado el respondente para activar el envío. Junto a cada condición puedes definir los links y archivos adjuntos a incluir.
