¿Qué es Enviar notificación?

Esta categoría genera alertas automáticas según las respuestas recibidas. Existen tres tipos:

  • Alerta de satisfacción — se genera cuando se acumula una cantidad configurada de notas positivas en una pregunta específica, dentro de un período en horas desde la primera respuesta.
  • Alerta de insatisfacción — se genera cuando se acumula una cantidad configurada de notas negativas en una pregunta específica, dentro de un período determinado.
  • Alerta personalizada — permite definir condiciones propias combinando preguntas, datos extras y filtros avanzados.
