¿Qué son las alertas personalizadas?

Las alertas personalizadas permiten configurar condiciones específicas combinando múltiples variables para generar notificaciones más precisas que las alertas estándar de satisfacción/insatisfacción.

Configuración

En la lista de notificaciones personalizadas, haz clic en el ícono de nueva acción y completa el formulario:

Figura 1 — Formulario de configuración de alerta personalizada.
Sección Campo Descripción
EncuestasEncuestaEncuesta a la que estará asociada la acción.
EncuestasCampañas / SensoresCampañas o sensores específicos a asociar (opcional).
EncuestasFiltros de datos extrasFiltros adicionales sobre la base de datos del encuestado.
CondicionesTipo condiciónDefine la lógica que activa la alerta.
DestinatariosUsuariosUsuarios de QServus que recibirán la notificación.
