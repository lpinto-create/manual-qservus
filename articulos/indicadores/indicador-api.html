¿Qué es el indicador de API?

El indicador de API pertenece al grupo de Datos operacionales y permite visualizar en los tableros de QServus datos externos enviados mediante la API de QServus. Es ideal para integrar métricas de sistemas externos (CRM, ERP, plataformas propias) directamente en los tableros de satisfacción.

Configuración

Campo Descripción
Nombre del indicadorNombre con el que se identificará el indicador en el tablero.
Ancho del indicadorEspacio que ocupará en el tablero: Estándar (12:4), Mitad horizontal (12:6), Doble en horizontal (12:8), Triple en horizontal (12:12).
MétricaPorcentaje — define cómo se representa el valor enviado desde la API.
Tipo de indicadorAcumulado o Evolutivo según cómo se quieran agrupar los datos recibidos.
Tipo de gráficoVaría según el tipo de indicador seleccionado.
Categorías de datosLista de categorías de datos configuradas para este indicador, con descripción, fecha de creación y acciones disponibles.
ℹ️
Para usar este indicador debes enviar datos al endpoint correspondiente de la API de QServus. Consulta la sección API para ver los detalles de integración.
