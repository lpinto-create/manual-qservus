¿Qué tipos de indicadores existen?

Al crear un nuevo indicador en un tablero, puedes elegir entre dos grupos según la fuente de datos.

Desde Datos de respuestas

Tipo Pregunta asociada
Indicador de escalaEscalas de caras, estrellas o números (1-3, 1-5, 1-7, 1-10 y 0-10).
Indicador de alternativasSelección con respuesta única, múltiple o tipo lista.
Indicador de textoTexto simple o multilínea.
Indicador de Sí/NoSolo para preguntas tipo Sí/No.
Indicador de matriz de doble entradaSolo para preguntas de matriz de doble entrada.
Indicador de tablaTablas dinámicas y tabla de respuestas.
Indicador de analítica avanzadaSolo para texto multilínea. Usa IA para analizar sentimiento y categorizar respuestas.

Desde Datos operacionales

Tipo Descripción
Indicador de APIIndicador alimentado con datos externos enviados mediante la API de QServus.
Indicador de métricasMuestra información de participación en encuestas y estadísticas de envíos.
💡
Al crear un indicador, el sistema te pedirá seleccionar el tipo antes de mostrar el formulario de configuración.
