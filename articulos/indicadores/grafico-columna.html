¿Cuándo se usa?

El gráfico de columnas se usa para representar preguntas de selección, escala, Sí/No y matrices. Es especialmente útil para comparaciones entre categorías o períodos de tiempo.

Cómo se lee

Cada columna representa una alternativa o período. La altura de la columna indica la cantidad o porcentaje de respuestas para esa categoría. Permite comparar visualmente múltiples valores de forma clara y rápida.
