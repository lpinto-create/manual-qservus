¿Qué es el tablero automático?

El tablero automático se genera de forma automática al crear una encuesta y comenzar a recopilar respuestas. Una vez creado, permite editar los indicadores para adaptarlo a las necesidades del cliente.

Es ideal para usuarios que están comenzando en la plataforma y necesitan una base de indicadores lista para usar sin configuración adicional.
