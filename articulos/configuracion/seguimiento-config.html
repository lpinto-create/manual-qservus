¿Qué es la configuración de seguimiento?

Desde aquí puedes habilitar o deshabilitar el uso de seguimientos para las respuestas que tienen una acción personalizada asociada. Al habilitarlo, aparecen opciones adicionales de configuración.

Figura 1 — Pantalla de configuración del módulo de seguimiento.
Campo Obligatorio Descripción
Template de correoSíTemplate que se usará para las notificaciones de seguimiento.
Texto de ayuda para el comentarioNoEnunciado del campo de comentario en el formulario de seguimiento.
Obligar comentario antes de finalizarSíSi está activo, el agente debe dejar un comentario antes de cerrar el seguimiento.
Habilitar formulario de cierreSíActiva un formulario adicional para registrar la conformidad del cliente al cerrar.
