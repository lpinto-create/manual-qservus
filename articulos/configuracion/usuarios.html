¿Para qué sirve?

La sección de usuarios permite definir qué personas tienen acceso a tu organización registrada en QServus. Desde aquí puedes agregar nuevos usuarios, editar los existentes y gestionar sus permisos según el perfil asignado.

Figura 10.23 — Gestión de usuarios del sistema

Agregar un usuario

Para agregar un nuevo usuario, haz clic en el ícono de creación en la lista de usuarios. Aparecerá el formulario solicitando los datos del nuevo usuario: nombre, apellido, correo electrónico y perfil.

Figura 10.23 — Formulario para crear un nuevo usuario

Importar y exportar usuarios

El sistema permite realizar carga masiva y descarga de usuarios usando archivos XLSX. Para acceder a estas opciones, usa los botones de importar y exportar disponibles en la lista de usuarios.

Figura 10.24 — Opciones de importación y exportación de usuarios
💡
Los permisos de cada usuario dependen del perfil que se le asigne. Para configurar perfiles, revisa el artículo Perfiles.
