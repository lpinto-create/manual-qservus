¿Qué es?

El reporte web genera automáticamente un sitio web con el informe de resultados de la encuesta, sensor o campaña seleccionada. El sitio tiene una URL única que puedes compartir con cualquier persona sin que necesite acceso a QServus.

Figura 7.6 — Reporte web

Cómo acceder

Desde Reportes → Reportes de encuestas, selecciona una encuesta y elige la opción Reporte web. El sistema genera la URL del sitio de resultados.

💡
El reporte web se actualiza automáticamente con nuevas respuestas. Comparte el enlace una sola vez y siempre mostrará los datos más recientes.
