¿Qué es el reporte web?

Esta opción genera automáticamente un sitio web con los resultados de la encuesta, sensor o campaña seleccionada. Cada reporte web tiene una URL única que puedes compartir con quienes necesiten revisar los resultados, sin necesidad de descargar ningún archivo.

Figura 1 — Reporte web con URL única para compartir.
💡
El reporte web es ideal para compartir resultados en tiempo real con equipos o clientes externos. Si necesitas un documento descargable, usa el Reporte en PDF.
