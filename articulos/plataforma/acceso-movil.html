Cómo acceder desde móvil

QServus no requiere instalación de ninguna aplicación móvil. El acceso se realiza directamente desde el navegador del dispositivo.

1

Abre el navegador en tu dispositivo

Usa Chrome (Android) o Safari (iOS) para la mejor experiencia.

2

Ingresa la URL de acceso

Navega a https://qservus.redcalidad.com o a la URL específica de tu organización.

3

Inicia sesión con tus credenciales

Ingresa tu correo electrónico y contraseña, luego presiona Ingresar.

Agregar acceso directo al escritorio (opcional)

Puedes crear un acceso directo en la pantalla de inicio del dispositivo para acceder a QServus con un solo toque:

  • Chrome (Android): menú (⋮) → "Agregar a pantalla de inicio"
  • Safari (iOS): ícono de compartir → "Agregar a pantalla de inicio"
ℹ️
Este acceso directo funciona como un marcador y abre QServus directamente en el navegador. No es una aplicación nativa.

Funcionalidades disponibles en móvil

  • Visualización de respuestas en tiempo real (¿Qué está pasando?)
  • Consulta de indicadores y tableros
  • Acceso al detalle de respuestas individuales
  • Revisión de seguimientos
💡
Para crear encuestas, configurar sensores o campañas, se recomienda usar un computador de escritorio. La interfaz de administración es funcional en móvil pero está optimizada para pantallas más grandes.
