Compatibilidad de dispositivos

La plataforma está diseñada para funcionar sin problemas en cualquier dispositivo, tanto móviles como de escritorio. Solo requiere una versión actualizada de los navegadores Chrome, Firefox, Safari o Microsoft Edge.

ℹ️
QServus es una aplicación web. No se requiere instalar ningún software ni aplicación adicional en el dispositivo.

¿Puedo responder encuestas desde móviles?

Sí. Las encuestas pueden responderse sin problemas desde dispositivos móviles. De hecho, una de cada dos encuestas de QServus se responden desde un tablet o smartphone.

El diseño de las encuestas se adapta automáticamente al tamaño de pantalla del dispositivo (diseño responsivo).

¿Puedo administrar la plataforma desde el móvil?

La interfaz de administración de QServus también es accesible desde dispositivos móviles. Sin embargo, para tareas complejas como la creación de encuestas o la configuración de tableros, se recomienda usar un computador de escritorio o laptop para una mejor experiencia.

Dispositivos compatibles

  • Computadores de escritorio y laptops — experiencia completa recomendada para administración
  • Tablets — compatible para administración y respuesta de encuestas
  • Smartphones — compatible para respuesta de encuestas y consulta de resultados
💡
Para la mejor experiencia de administración, usa QServus desde un computador con pantalla de al menos 1280px de ancho.
