¿Qué es la API de QServus?

QServus expone una API REST que permite a sistemas externos interactuar con la plataforma de forma programática. Mediante la API puedes automatizar el envío de encuestas, registrar respuestas desde tus propios sistemas, obtener listados de sensores activos y consultar el estado de la plataforma.

Formato de datos

Para las entradas y salidas de la API se utiliza el formato JSON (JavaScript Object Notation). JSON es un formato liviano para el intercambio de datos que describe la información con una sintaxis estructurada de clave-valor, compatible con la mayoría de los lenguajes de programación.

Endpoints disponibles

La API de QServus incluye los siguientes endpoints principales:

  • Autenticación: obtiene el token de acceso necesario para usar el resto de los endpoints.
  • Obtener sensores activos: devuelve la lista de sensores activos con sus encuestas y preguntas.
  • Recibir respuesta: registra una o varias respuestas de clientes en el sistema.
  • Enviar encuesta a destinatario: envía una encuesta a un destinatario específico.
  • Obtener destinatario en lista negra: consulta si un correo está en la lista negra.
  • Verificar estado de la plataforma: comprueba que los servicios estén operativos.
💡
Para comenzar a usar la API, primero debes autenticarte para obtener tu token. Luego puedes usar ese token en el encabezado de cada solicitud a los endpoints disponibles.
